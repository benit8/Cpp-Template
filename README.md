# Project
C++ template project

Includes:
- CMake lists
- Command-line option parser
- Unit tests

## Build
```sh
$ ./do
```
or
```sh
$ ./do build
```

## Run
```sh
$ ./do run
```

Arguments after `run` will get passed to the binary.

## Tests
This project includes unit tests, done with [Criterion](https://github.com/Snaipe/Criterion).

```sh
$ ./do test
```
